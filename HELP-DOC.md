## Getting Started

### Welcome to EaseMail

EaseMail is a modern email client that connects to your Microsoft 365 account. It provides a clean, intuitive interface for managing your emails, calendar, and contacts all in one place.

### Initial Setup

1. **Visit EaseMail**: Go to the EaseMail homepage and click "Sign In" or "Get Started"
2. **Sign In with Microsoft**: Click "Sign in with Microsoft 365" button on the login page
3. **Authenticate**: Log in with your Microsoft 365 account credentials
4. **Grant Permissions**: Authorize EaseMail to access your emails, calendar, and contacts
5. **Wait for Sync**: The initial sync will download your recent emails (last 30 days by default)
6. **Start Using**: Once sync completes, you're ready to start managing your emails!

**For Teams:** If someone from your organization has already signed up, you'll automatically join their workspace when you sign in with your company email domain.

### System Requirements

- Modern web browser (Chrome, Edge, Firefox, Safari)
- Microsoft 365 account (personal or work/school)
- Internet connection

## Email Features

### Composing Emails

**How to Compose:**
1. Click the **Compose** button in the top bar (or press **C**)
2. Enter recipient email addresses (press Enter or comma to add multiple)
3. Add CC/BCC recipients if needed
4. Enter a subject line
5. Type your message in the editor
6. Use the formatting toolbar to style text (bold, italic, lists, etc.)
7. Add attachments by clicking the paperclip icon
8. Select a signature from the dropdown (if configured)
9. Click **Send** (or press **Cmd/Ctrl+Enter**)

**Composer Features:**
- Auto-save as draft
- Rich text formatting (bold, italic, underline, lists, links)
- File attachments
- Multiple recipients (To, CC, BCC)
- Signature insertion
- Undo close protection (warns if unsaved changes)

**Keyboard Shortcuts in Composer:**
- `Cmd/Ctrl+Enter` - Send email
- `Cmd/Ctrl+B` - Bold text
- `Cmd/Ctrl+I` - Italic text
- `Cmd/Ctrl+U` - Underline text
- `Escape` - Close composer

### Reading Emails

**Viewing an Email:**
1. Click any email in the message list
2. The email opens in the right pane (message viewer)
3. Use the toolbar actions to reply, forward, archive, or delete

**Message Viewer Features:**
- Full email content (HTML and plain text)
- Sender details with contact icon
- Attachments download
- Inline images
- Reply/Reply All/Forward buttons
- Archive and Delete actions
- Flag/Star for important emails
- Undo support for archive/delete (5-second window)

**Re-Sync Feature:**
If an email shows only a preview, click the **Re-Sync** button to download the full email content from Microsoft 365.

**Contact Slide-In:**
Click the contact icon in any email header to view:
- Contact information (phone, company, location)
- Recent email history with this contact
- Quick compose to contact

### Folders and Organization

**Default Folders:**
- **Inbox** - New incoming messages
- **Sent Items** - Emails you've sent
- **Drafts** - Unfinished emails (auto-saved)
- **Deleted Items** - Trash folder
- **Archive** - Archived messages
- **Junk Email** - Spam folder

**Custom Folders:**
All folders from your Microsoft 365 account sync automatically and appear in the sidebar below the default folders.

**Folder Actions:**
- Click any folder to view its contents
- Folder counts update in real-time
- Use **E** key to archive messages (moves to Archive folder)
- Use **Backspace/Delete** to move to trash

### Search

**How to Search:**
1. Click the search bar in the top navigation (or press **Cmd/Ctrl+K**)
2. Type your search query
3. Results appear as you type
4. Use **↑** and **↓** arrow keys to navigate results
5. Press **Enter** to open the selected email
6. Press **Escape** to close search

**Search Capabilities:**
- Searches sender names, email addresses, subjects, and body content
- Real-time results as you type
- Highlights matching text
- Shows email previews
- Indicates unread and attachment status

### Email Actions

**Reply / Reply All / Forward:**
- **Reply** - Respond to the sender only
- **Reply All** - Respond to all recipients
- **Forward** - Send the email to new recipients

**Archive:**
- Moves email to Archive folder
- 5-second undo window
- Keyboard shortcut: **E**

**Delete:**
- Moves email to Deleted Items folder
- 5-second undo window
- Keyboard shortcut: **Backspace** or **Delete**

**Flag/Star:**
- Mark emails as important
- Quick access to flagged emails
- Keyboard shortcut: **S**

**Mark as Read/Unread:**
- Emails are automatically marked as read when opened
- Unread emails show a blue dot indicator

## Calendar

### Creating Events

**How to Create an Event:**
1. Click the **Event** button in the top bar (next to Compose)
2. Enter event title
3. Set start and end date/time
4. Add a location (optional)
5. Add attendees by typing email addresses
6. Toggle **All Day** for all-day events
7. Toggle **Online Meeting** to create a Teams meeting
8. Add a description
9. Set a reminder (15 min, 30 min, 1 hour, etc.)
10. Click **Create Event**

**Event Features:**
- Single and recurring events
- All-day events
- Online meetings (Teams integration)
- Multiple attendees with invitations
- Reminders
- Event descriptions
- Location information

### Viewing Your Calendar

**Calendar Views:**
- **Month View** - See the whole month at a glance
- **Week View** - Detailed week schedule
- **Day View** - Hour-by-hour view of a single day
- **Agenda View** - List of upcoming events

**Navigation:**
- Use arrow buttons to move forward/backward
- Click **Today** to jump to the current date
- Click any event to view details

**Calendar Sync:**
- Events sync bidirectionally with Microsoft 365
- Changes made in EaseMail appear in Outlook and vice versa
- Meeting invites are sent automatically

## Contacts

### Viewing Contacts

**Contact Slide-In (from Email):**
1. Open any email
2. Click the contact icon in the message header
3. View contact details and email history

**Contacts Page:**
1. Navigate to **Contacts** from the sidebar
2. Browse all contacts
3. Use the search bar to find specific contacts
4. Click any contact to view full details

### Managing Contacts

**Creating a Contact:**
1. Go to Contacts page
2. Click **New Contact**
3. Fill in contact information:
   - Name (first, last, middle, nickname)
   - Email address (required)
   - Phone numbers (mobile, home, business)
   - Company and job title
   - Address (street, city, state, postal code, country)
   - Birthday and anniversary
   - Notes
4. Click **Save**

**Contact Features:**
- Sync with Microsoft 365 contacts
- Auto-inferred from email interactions
- Contact groups for organization
- Mark as favorite for quick access
- Email history tracking

## Settings

### Managing Connected Accounts

**View Account Status:**
1. Navigate to **Settings** from sidebar
2. View all connected Microsoft 365 accounts
3. Check sync status, last sync time, and message counts

**Account Information Displayed:**
- Email address
- Display name
- Account type (Work or Personal)
- Status (Active, Syncing, Error, Disconnected)
- Last full sync timestamp
- Total message count
- Unread count
- Folder count

**Reconnecting an Account:**
If an account shows "Error" or "Disconnected" status:
1. Click **Reconnect** button
2. Sign in with your Microsoft 365 account
3. Grant permissions again
4. Wait for sync to resume

### Email Signatures

**Creating a Signature:**
1. Go to **Settings** > **Signatures** tab
2. Click **New Signature**
3. Enter a name for your signature
4. Type or paste your signature content
5. Select which account this applies to:
   - **All Accounts** - Used across all connected accounts
   - **Specific Account** - Only for selected account
6. Check **Set as Default** to use automatically in new emails
7. Click **Save**

**Signature Features:**
- Multiple signatures per account
- Default signature auto-inserted when composing
- Account-specific signatures override global signatures
- Basic HTML formatting supported
- Easy editing and deletion

**Using Signatures:**
- Default signatures are inserted automatically
- Change signature from dropdown in composer
- Remove signature if not needed

### Sync & Performance

**How Sync Works:**
- **Initial Sync**: Downloads recent emails (last 30 days)
- **Incremental Sync**: New emails sync every few minutes automatically
- **Calendar Sync**: Events sync in the background
- **Contacts Sync**: Contacts sync automatically

**Manual Re-Sync:**
- Use the **Re-Sync** button on individual emails to download full content
- System automatically handles sync - no manual refresh needed

**Performance Tips:**
- Clear browser cache if the app feels slow
- Use Chrome or Edge for best performance
- Keep your browser updated
- Large mailboxes (>10,000 emails) may take longer to search

## Troubleshooting

### Common Issues

#### Emails Not Syncing

**Symptoms**: New emails don't appear, sync indicator stuck

**Solutions**:
1. Check account status in **Settings** > **Accounts**
2. If status shows "Error", click **Reconnect**
3. Ensure you have internet connection
4. Check Microsoft 365 service status
5. Try logging out and back in

#### Missing Email Content

**Symptoms**: Email shows only preview text, not full content

**Solution**:
1. Click the **Re-Sync** button on the message
2. This downloads the full email body from Microsoft 365
3. Large emails may take a few seconds

#### Signatures Not Appearing

**Symptoms**: Signature not inserted when composing

**Solutions**:
1. Go to **Settings** > **Signatures**
2. Ensure you have at least one signature created
3. Check that a signature is set as **Default**
4. For specific accounts, ensure account-specific signature is configured
5. Try selecting signature manually from dropdown in composer

#### Search Not Working

**Symptoms**: Search returns no results or is slow

**Solutions**:
1. Try refreshing the page (search indexes build in background)
2. Ensure you're searching in the correct account
3. Wait a few minutes after initial setup for search index to build
4. Check spelling of search terms

#### Calendar Events Not Showing

**Symptoms**: Events don't appear on calendar page

**Solutions**:
1. Navigate to the Calendar page to trigger a sync
2. Check account status in Settings
3. Ensure calendar permissions were granted during login
4. Try reconnecting the account

### Reconnecting Your Account

If you experience persistent issues, reconnecting often resolves them:

1. Go to **Settings** > **Accounts**
2. Find the problematic account
3. Click **Reconnect**
4. Sign in with your Microsoft 365 credentials
5. Grant all requested permissions
6. Wait for sync to complete

### Performance Issues

**If EaseMail feels slow:**

1. **Clear Browser Cache**:
   - Chrome: Settings → Privacy → Clear browsing data
   - Edge: Settings → Privacy → Choose what to clear
   - Firefox: Options → Privacy → Clear Data

2. **Update Your Browser**:
   - Use the latest version of Chrome, Edge, Firefox, or Safari
   - Older browsers may have performance issues

3. **Check System Resources**:
   - Close unnecessary browser tabs
   - Ensure adequate RAM available
   - Check CPU usage

4. **Large Mailboxes**:
   - Mailboxes with >10,000 emails may be slower
   - Search and folder operations take longer
   - Consider archiving old emails

### Getting Help

If you continue to experience issues:

1. Check this help documentation for solutions
2. Try the troubleshooting steps above
3. Restart your browser
4. Try a different browser
5. Check the Feature Update Log below for known issues
